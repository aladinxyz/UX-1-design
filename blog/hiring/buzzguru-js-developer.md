**Про наш стек**

Используем:
- стек технолгий: React, node.js , RN
- Почти последний Babel с декораторами и элвис операторами
- React 16.5 (есть уже ветка с 16.7, там где хуки, но еще она не очень стабильна.)  
- MobX - но от него только Provider и inject по сути (аналоги Provider, connect в ридакс)
- Подключен PostCSS, но в последнее время его почти не используем
- Emotion (это аналог styled components) для css in js
- ну и так исторически сложилось, что прикручен бутстрап (для сетки и модалок), но в скором времени постараемся от него отказаться
- для некоторых контролов используем Ant design
- Управление состоянием формы с помощью Formik
- BS4 + reactstrap

Помимо этого, есть еще сам стартер кит, собранный из знаменитого @kriasoft: React Starter Kit (это если интересно, на чем бекэнд)
